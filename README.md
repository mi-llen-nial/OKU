# OKU

**OKU** — персонализированная платформа экзаменов (RU/KZ): генерация тестов с помощью AI, проверка ответов, рекомендации по темам и аналитика для школ и преподавателей

**Стек:** FastAPI · PostgreSQL · Redis/RQ · Next.js 14 · Docker/Caddy.

Один репозиторий: **`backend/`** (REST API `/api/v1`), **`frontend/`** (сайт и приложение; режимы по hostname), при необходимости **`mobile/`**. Типовой прод-стек описан в `docker-compose.yml`.

Подробности — запуск, переменные окружения, деплой, API, демо-аккаунты, импорт каталога и **архитектура кода** — в **[project.md](./project.md)**.

**Быстрый старт локально:** `make setup`, затем в отдельных терминалах `make backend` и `make frontend` (детали в `project.md`).
